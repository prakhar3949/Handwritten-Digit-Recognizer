# Handwritten Digit Recognizer

Handwritten digit recognition using opencv, SVM and Python

## Dependencies
1. skimage
2. opencv
3. PIL
4. numpy
5. tkinter

## Usage
1. Clone/download the repository
2. Run app.py script
3. Drag the mouse to draw number(s)
